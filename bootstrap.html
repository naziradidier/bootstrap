Menu simple

Voici un exemple de menu avec 4 liens.

  1. <ul class="nav">

  2. <li class="nav-item">

  3. <a class="nav-link active" href="#">Accueil</a>

  4. </li>

  5. <li class="nav-item">

  6. <a class="nav-link" href="#">A propos</a>

  7. </li>

  8. <li class="nav-item">

  9. <a class="nav-link" href="#">Services</a>

  10. </li>

  11. <li class="nav-item">

  12. <a class="nav-link" href="#">Contact</a>

  13. </li>

  14. </ul>





Menu simplifié

Il est possible d’alléger le code en supprimant la balise <ul> et travailler avec la sémantique nav.

  1. <nav class="nav">

  2. <a class="nav-link active" href="#">Accueil</a>

  3. <a class="nav-link" href="#">A propos</a>

  4. <a class="nav-link" href="#">Services</a>

  5. <a class="nav-link" href="#">Contact</a>

  6. </nav>

Sémantiquement la balise <nav> suffit pour annoncer une zone de navigation.

Style personnalisé

Vous pouvez utiliser les classes de style abordées dans le chapitre « Gérer les alignements » pour :

  • centrer les liens avec justify-content-center

  • aligner à droite les liens avec justify-content-end

  1. <nav class="nav justify-content-center"> <!-- alignement centre -->

  2. <a class="nav-link active" href="#">Accueil</a>

  3. <a class="nav-link" href="#">A propos</a>

  4. <a class="nav-link" href="#">Services</a>

  5. <a class="nav-link" href="#">Contact</a>

  6. </nav>

  7. <hr>

  8. <nav class="nav justify-content-end"> <!-- alignement droite -->

  9. <a class="nav-link active" href="#">Accueil</a>

  10. <a class="nav-link" href="#">A propos</a>

  11. <a class="nav-link" href="#">Services</a>

  12. <a class="nav-link" href="#">Contact</a>

  13. </nav>

  1. <nav class="nav flex-column">

  4. <a class="nav-link" href="#">Services</a>

  5. <a class="nav-link" href="#">Contact</a>

  6. </nav>



Menu à onglets

Voici une variante de menu basée sur les onglets et utilisant la classe nav-tabs . Ce type de menu est souvent utilisé avec un composant javascript (voir le chapitre « les composants »).

  1. <nav class="nav nav-tabs">

  2. <a class="nav-link active" href="#">Accueil</a>

  3. <a class="nav-link" href="#">A propos</a>

  4. <a class="nav-link" href="#">Services</a>

  5. <a class="nav-link" href="#">Contact</a>

  6. </nav>



Menu bouton

Les liens peuvent être présentés sous forme de bouton avec nav-pills .

  1. <nav class="nav nav-pills">

  2. <a class="nav-link active nav-item" href="#">Accueil</a>

  3. <a class="nav-link" href="#">A propos</a>

  4. <a class="nav-link" href="#">Services</a>

  5. <a class="nav-link" href="#">Contact</a>

  6. </nav>



Bouton large

Il est possible de répartir la taille des boutons sur la largeur du conteneur avec nav-fill . Les liens apparaîtront plus larges. Il faut néanmoins inclure nav-item dans chaque lien.

  1. <nav class="nav nav-pills nav-fill">

  2. <a class="nav-link active nav-item" href="#">Accueil</a>

  3. <a class="nav-link nav-item" href="#">A propos</a>

  4. <a class="nav-link nav-item" href="#">Services</a>

  5. <a class="nav-link nav-item" href="#">Contact</a>

  6. </nav>

Mode Responsive

Vous pouvez exploiter les points de rupture (sm, md, lg, xl) pour adapter le menu en fonction de la taille de l’appareil.

Le code ci-dessous propose deux présentation du même menu. Pour une taille d’écran inférieur à 576px les boutons sont empilés ( flex-column ). Mais au delà de cette taille, les liens s’alignent flex-sm-row .

  1. <nav class="nav nav-pills flex-column flex-sm-row">

  2. <a class="nav-link active" href="#">Accueil</a>

  3. <a class="nav-link" href="#">A propos</a>

  4. <a class="nav-link" href="#">Services</a>

  5. <a class="nav-link" href="#">Contact</a>

  6. </nav>



Menu déroulant

Les listes déroulantes sont construites avec la bibliothèque tierce Popper.js que vous avez normalement chargée avec Jquery et Bootstrap

Le lien déroulant

Un lien déroulant affiche une liste de liens lorsqu’on clique dessus. Cette liste disparaît si vous cliquez n’importe où sur la page ou si vous cliquez à nouveau sur le bouton.

  1. <div class="dropdown">

  2. <a class="nav-link dropdown-toggle" data-toggle="dropdown" href="#" role="button" aria-haspopup="true" aria-expanded="false">Services</a> <!-- lien -->

  3. <div class="dropdown-menu">

  4. <a class="dropdown-item" href="#">Service 1</a>

  5. <a class="dropdown-item" href="#">Autre service</a>

  6. <a class="dropdown-item" href="#">Dernier Service</a>

  7. </div>

  8. </div>



Le bouton déroulant

C’est une variante du lien déroulant avec la balise <button> à la place de la balise <a>.

  1. <div class="dropdown">

  2. <button class="nav-link dropdown-toggle" data-toggle="dropdown" href="#" role="button" aria-haspopup="true" aria-expanded="false">Services</button> <!-- bouton -->

  3. <div class="dropdown-menu">

  4. <a class="dropdown-item" href="#">Service 1</a>

  5. <a class="dropdown-item" href="#">Autre service</a>

  6. <a class="dropdown-item" href="#">Dernier Service</a>

  7. </div>

  8. </div>





Le bouton déroulant partagé

On regroupe le bouton texte (inactif) et le bouton curseur (sur lequel on clique pour dérouler) pour former un seul bouton btn-group.

  1. <div class="btn-group"> <!-- groupe de boutons -->

  2. <button type="button" class="btn">Services</button>

  3. <button type="button" class="btn dropdown-toggle dropdown-toggle-split" data-toggle="dropdown" aria-haspopup="true" aria-expanded="false">

  4. <span class="sr-only">Toggle Dropdown</span>

  5. </button>

  6. <div class="dropdown-menu">

  7. <a class="dropdown-item" href="#">Service 1</a>

  8. <a class="dropdown-item" href="#">Autre service</a>

  9. <a class="dropdown-item" href="#">Dernier service</a>

  10. </div>

  11. </div>





Entête de menu

Vous pouvez ajoutez des entête pour créer des sections de sous-menu avec dropdown-header .

  1. <div class="btn-group">

  2. <button type="button" class="btn">Services</button>

  3. <button type="button" class="btn dropdown-toggle dropdown-toggle-split" data-toggle="dropdown" aria-haspopup="true" aria-expanded="false">

  4. <span class="sr-only">Toggle Dropdown</span>

  5. </button>

  6. <div class="dropdown-menu">

  7. <h6 class="dropdown-header">Accompagnement</h6> <!-- entête -->

  8. <a class="dropdown-item" href="#">Service 1</a>

  9. <a class="dropdown-item" href="#">Autre service</a>

  10. <a class="dropdown-item" href="#">Dernier service</a>

  11. </div>

  12. </div>

  4. <span class="sr-only">Toggle Dropdown</span>

  5. </button>

  6. <div class="dropdown-menu">

  7. <h6 class="dropdown-header">Accompagnement</h6>

  13. </div>

  14. </div>

Le menu déroulant simple

Il suffit d’intégrer le bloc déroulant ( dropdown-menu ) dans un menu simple ( nav ).

  1. <nav class="nav"> <!-- menu simple -->

  2. <a class="nav-link active" href="#">Accueil</a>

  3. <a class="nav-link dropdown-toggle" data-toggle="dropdown" href="#" role="button" aria-haspopup="true" aria-expanded="false">Services</a>

  4. <div class="dropdown-menu"> <!-- bloc déroulant -->

  5. <a class="dropdown-item" href="#">Service 1</a>

  6. <a class="dropdown-item" href="#">Autre service</a>

  7. <a class="dropdown-item" href="#">Dernier Service</a>

  8. </div>

  9. <a class="nav-link" href="#">A propos</a>

  10. <a class="nav-link" href="#">Services</a>

  11. <a class="nav-link" href="#">Contact</a>

  12. </nav>





La barre est crée avec navbar.

  1. <nav class="navbar navbar-light bg-light">

  2. </nav>

Le point de rupture (mode mobile) est définit par navbar-expand-* . Remplacez * par sm, md, lg ou xl.

  1. <nav class="navbar navbar-light bg-light navbar-expand-lg">

  2. </nav>

Affichez le logo ou la marque avec navbar-brand.

  1. <nav class="navbar navbar-light bg-light navbar-expand-lg">

  2. <a class="navbar-brand" href="#">Logo</a>

  3. </nav>

Placez ensuite le bouton navbar-toggler et son icône barre navbar-toggler-icon pour le mode mobile.

  1. <nav class="navbar navbar-light bg-light navbar-expand-lg">

  2. <a class="navbar-brand" href="#">Logo</a>

  3. <button class="navbar-toggler" type="button" data-toggle="collapse" data-target="#navbarSupportedContent" aria-controls="navbarSupportedContent" aria-expanded="false" aria-label="Toggle navigation">

  4. <span class="navbar-toggler-icon"></span>

  5. </button>

  6. </nav>

Intégrez ensuite les liens de navigations dans un bloc navbar-collapse .

  1. <nav class="navbar navbar-light bg-light navbar-expand-lg">

  2. <a class="navbar-brand" href="#">Logo</a>

  3. <button class="navbar-toggler" type="button" data-toggle="collapse" data-target="#navbarSupportedContent" aria-controls="navbarSupportedContent" aria-expanded="false" aria-label="Toggle navigation">

  4. <span class="navbar-toggler-icon"></span>

  5. </button>

  6. <div class="collapse navbar-collapse" id="navbarSupportedContent">

  7. <ul class="navbar-nav mr-auto">

  8. <li class="nav-item active">

  9. <a class="nav-link" href="#">Accueil</a>

  10. </li>

  11. <li class="nav-item">

  12. <a class="nav-link" href="#">A propos</a>

  13. </li>

  14. <li class="nav-item">

  15. <a class="nav-link" href="#">Services</a>

  16. </li>

  17. <li class="nav-item">

  18. <a class="nav-link" href="#">Contact</a>

  19. </li>

  20. </ul>

  21. </div>

  22. </nav>



Avec menu déroulant

Il suffit d’intégrer le bloc déroulant ( dropdown-menu ) dans un menu.

  1. <nav class="navbar navbar-expand-lg navbar-light bg-light">

  2. <a class="navbar-brand" href="#">Logo</a>

  3. <button class="navbar-toggler" type="button" data-toggle="collapse" data-target="#navbarSupportedContent" aria-controls="navbarSupportedContent" aria-expanded="false" aria-label="Toggle navigation">

  4. <span class="navbar-toggler-icon"></span>

  5. </button>

  6. <div class="collapse navbar-collapse" id="navbarSupportedContent">

  7. <ul class="navbar-nav mr-auto">

  8. <li class="nav-item active">

  9. <a class="nav-link" href="#">Accueil</a>

  10. </li>

  11. <li class="nav-item">

  12. <a class="nav-link" href="#">A propos</a>

  13. </li>

  14. <li class="nav-item dropdown">

  15. <a class="nav-link dropdown-toggle" href="#" id="navbarDropdownMenuLink" data-toggle="dropdown" aria-haspopup="true" aria-expanded="false">

  16. Services

  17. </a>

  18. <!-- bloc menu déroulant -->

  19. <div class="dropdown-menu" aria-labelledby="navbarDropdownMenuLink">

  20. <a class="dropdown-item" href="#">Service 1</a>

  21. <a class="dropdown-item" href="#">Autre service action</a>

  22. <a class="dropdown-item" href="#">Dernier service</a>

  23. </div>

  24. </li>

  25. <li class="nav-item">

  26. <a class="nav-link" href="#">Contact</a>

  27. </li>

  28. </ul>

  29. </div>

  30. </nav
